# 产品设计：A/H 股长线持仓研究助手

## 1. 核心用户场景

用户每月、每季度、每年度对自己的持仓和观察池做复盘：

- 月度：看价格、趋势、风险、公告、政策；
- 季度：看财报、经营质量、现金流、投资假设是否被证伪；
- 年度：重建股票池，复盘规则有效性，修正权重。

## 2. 核心页面

### 2.1 首页看板

展示：

- A股候选池数量；
- 港股候选池数量；
- 持仓健康状态；
- 本月新增风险；
- 本月新增观察对象；
- 需要复盘的股票。

### 2.2 数据准备页

展示：

- 行情数据更新时间；
- 财报数据更新时间；
- 股票基础信息更新时间；
- 数据缺失情况；
- 数据源状态；
- 导入/重建按钮。

### 2.3 股票池筛选页

支持：

- 市场：A股/港股；
- 行业；
- 市值；
- 分红；
- ROE；
- 估值分位；
- 250日相对强弱；
- 风险剔除状态。

### 2.4 单股研究页

展示：

- K线；
- 基本信息；
- 财务摘要；
- 估值分位；
- 评分雷达；
- 风险提示；
- 投资假设；
- Agent 分析结论。

### 2.5 持仓体检页

展示：

- 用户持仓；
- 成本；
- 当前盈亏；
- 长线评级；
- 建议动作；
- 下次观察点。

### 2.6 报告中心

支持：

- 月度持仓体检报告；
- 季度财报复盘报告；
- 年度股票池重建报告；
- 单股深度分析报告。

## 3. 决策等级

| 等级 | 说明 |
|---|---|
| HOLD | 持有 |
| WATCH | 加入观察池 |
| CAUTIOUS_HOLD | 谨慎持有 |
| REDUCE_EXIT | 减仓/撤出 |
| EXCLUDE | 剔除 |

## 4. 产品边界

- 不承诺收益；
- 不做短线预测；
- 不做自动交易；
- 不做无证据链荐股；
- 不用 LLM 编造数据。
