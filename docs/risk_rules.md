# 风险规则设计

## 1. 风险优先原则

长线投资先排雷，再筛优。

任何股票如果触发重大风险，应该先进入 `EXCLUDE` 或 `CAUTIOUS_HOLD`，而不是继续用估值便宜解释。

## 2. 硬性剔除规则

A股：

- ST / *ST；
- 退市整理；
- 长期停牌；
- 上市时间过短且财务数据不足；
- 非标审计意见；
- 连续多年亏损且无明确反转证据；
- 经营现金流长期严重为负；
- 财务造假或重大监管处罚。

港股：

- 长期停牌；
- 成交额严重不足；
- 非港股通且流动性差；
- 财报披露异常；
- 高度依赖单一事件反转；
- 大比例折价供股/频繁融资摊薄。

## 3. 谨慎持有规则

触发以下信号时，持仓应进入谨慎状态：

- ROE 连续下降；
- 毛利率明显下滑；
- 净利润增长但经营现金流恶化；
- 应收账款增速显著高于收入增速；
- 资产负债率快速上升；
- 大股东持续减持；
- 行业政策出现明显逆风；
- 250 日相对行业显著跑输。

## 4. 减仓/撤出规则

当投资假设被证伪时，应考虑减仓或撤出：

- 原本的成长逻辑消失；
- 原本的高分红不可持续；
- 现金流质量恶化并持续两个季度以上；
- 管理层承诺长期落空；
- 估值仍高但基本面下修；
- 风险事件无法量化或无法跟踪。
